# 📡 サブネット計算ツール（Subnet Calculator GUI）

このアプリケーションは、IPアドレスとCIDRに関する情報を簡単に計算・確認できる **GUIベースのサブネット計算ツール** です。IPv4 / IPv6 両対応で、ネットワーク設計や学習に役立ちます。

---

## 🖥️ 特徴

- ✅ 機器数からCIDRを自動算出
- ✅ CIDRから利用可能ホスト数を逆算
- ✅ IPv4 / IPv6 両対応
- ✅ ネットワーク情報の詳細表示
- ✅ エラーハンドリングと入力補助
